### Issue: 학과 구조화 시 단과대학명이 잘리는 현상 발생

PDF -> 텍스트 변환 과정에서 단과대학 이름이 줄바꿈 단위로 분리되어 
LLM이 "인문사회융합대학"을 "인문", "사회", "융합", "대학"처럼 독립된 섹션으로 오판하는 문제가 발생한다.

#### 예시 입력:
인문
사회
융합
대학
경영학과 35 18 48 …

#### 예시 출력 문제:
"college": "인문"

#### 원인:
- PDF 텍스트 추출 시 개행 단위 분리
- LLM이 복합명사를 줄 단위로 오해하는 패턴 오류

#### 해결 방향:
1. 미리 문자열 정제: 줄바꿈 제거 및 단과대 명칭 수동 복원
2. 프롬프트에 "단과대학 이름은 공백 없이 하나의 문자열"이라는 규칙 추가
3. JSON 후처리에서 검증 로직 적용

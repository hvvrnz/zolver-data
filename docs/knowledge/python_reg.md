하는 함수 생성을 위해 정규표현식을 이용함. 
모든 문자를 의미하는 '.'은 줄바꿈 및 개행문자(== \n)를 인식하지 못함.

# 1. [ ] : 문자 클래스
[abc] : a 또는 b 또는 c 중 하나 (문자 하나만 선택)
[ab|c] : ab 또는 c

## 문자 클래스 [...] 안에 띄어쓰기 여부
가능은 한데, [ab ] 이렇게 쓰면 의미는.
'a', 'b', ' ' ← 스페이스(띄어쓰기)
즉, [ab ]는 "a" 또는 "b" 또는 " " 중 하나
그렇기 때문에 띄어쓰기를 넣으면 문자 하나로서 공백 1개를 허용한다는 의미가 됨.

## 결론
띄어쓰기 넣으면 스페이스라는 문자도 매칭 대상에 포함
but 문법적으로는 허용

# 2. \s (공백문자), \S(공백이 아닌 문자)
## 2-1. \s — 모든 공백 문자
(1) 스페이스 " "
(2) 탭 \t
(3) 줄바꿈 \n
(4) 캐리지리턴 \r
(5) 폼피드 \f

## 2-2. \S - 공백(space) 이 아닌 문자
(1) 줄바꿈 \n
(2) 모든 문자!
